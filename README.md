# hand-vue-seed

> 汉得移动种子工程

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run watch

# Package release development environment, generate dist folder
npm run build-test

# Package release production environment, remove the log, sourcemap, and add gzip, generate prod folder
npm run build-prod

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run eslint
npm run lint
# run all tests
npm test
```
